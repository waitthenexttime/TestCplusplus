# 常用函数

## 数组最大值
```c++
vector<int> p = {2, 1, 3, 5, 4};
vector<int>::iterator a = max_element(p.begin(), p.end());
```
返回的是迭代器，寻值需要`*a`

## 数组最大值的索引
```c++
vector<int> p = {2, 1, 3, 5, 4};
vector<int>::iterator a = max_element(p.begin(), p.end());
cout<<a-p.begin();
```

## 字符串倒置

```c++
#include <cstring>
#include <algorithm>
int main()
{
      string str1;
      cin>>str1;
      cout<<"原始字符串是："<<str1<<endl;
      reverse(str1.begin(),str1.end());
      cout<<"转换后的字符串是："<<str1<<endl;
      return 0;
}
```

string num;

## Pair

戏称对子，可以类似看作有2个元素的结构

```c++
pair<int, char> a(1, 'e');
cout << a.first;
cout << a.second;
```
